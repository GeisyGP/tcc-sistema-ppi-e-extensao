# tcc-sistema-ppi-e-extensao
Sistema Web para Registro e Controle das Práticas Profissionais Integradas e Extensão - Trabalho de Conclusão de Curso

### Pré-requisitos

- [Node.js](https://nodejs.org/en/download/prebuilt-installer)
- [Docker compose](https://docs.docker.com/compose/install/)

## Passos inicias

Clone o repositorio com o seguinte comando:
```
git clone https://github.com/GeisyGP/
```

## Configurando o ambiente
Acesse o diretório em sua IDE de preferencia e insira o seguinte comando para instalar as dependencias:
```
npm install

npm run husky:init
```

Esse repositório contém o frontend e backend da aplicação, instruções específicas para cada um podem ser acessadas em seus respectivos diretórios.

- [Backend](/backend/README.md)
- [Frontend](/frontend/README.md)
